# CircularSlider
Circular Slider Component in React Native

## Setup

```shell
# Using yarn
yarn

# Using NPM
npm install

# Setup pods
npx pod-install

# Start metro bundler
yarn start

# Android
yarn android

# iOS
yarn ios
```

## Libraries Used

### react-native-reanimated
### react-native-gesture-handler
### react-native-svg
